Problems
========

Solutions to some programming problems. So far:

	* Sphere Online Judge (SPOJ)
	* Universidad de Valladolid Online Judge (UVA)
	* Caribbean Online Judge (COJ)
	* Facebook Hackercup
	* Google Codejam
	* Programando.org

Please don't copy and paste, use them as a guide.
